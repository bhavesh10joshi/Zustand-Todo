# Zustand Todo App 📝

A lightweight, frontend-only Todo application built to explore and understand global state management using **Zustand**. 

This project serves as a hands-on practice repository to learn how Zustand handles state outside of standard React hooks, keeping the codebase clean and avoiding prop drilling.

## 🚀 Tech Stack

- **Framework:** React (Functional Components)
- **Language:** TypeScript
- **State Management:** Zustand
- **Styling:** Tailwind CSS

## ✨ Features

- **Add Tasks:** Easily add new todos to the list.
- **Mark as Complete:** Toggle tasks between active and completed states (with visual text strikethrough).
- **Delete Tasks:** Remove tasks completely from the state.
- **Clean UI:** A dark-mode inspired, terminal-like user interface.
- **Custom Scrolling:** Implemented a hidden scrollbar for a sleeker look.

## 🧠 What I Learned (Zustand Focus)

Building this project helped solidify the following Zustand concepts:

- Creating a global store using the `create` function.
- Defining state types and actions in TypeScript for better developer experience and type safety.
- Reading and updating state cleanly from within components without needing complex `Providers` or wrappers.
- Separating business logic (state updates) from the UI layer.
